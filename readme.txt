== Installation ==

1. Install as a regular WordPress plugin
2. You have to use shortcode for generate alerts  in post or page. Example shortcode is given below.
[alearts class=warning"" text="<strong>Warning</strong> Best check yo self youre not looking too good."]

* You can change text.
* You can change class.
